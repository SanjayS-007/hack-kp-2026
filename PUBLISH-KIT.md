## 🏆 DEVPOST / HACKATHON PORTAL

**Elevator pitch (≤200 chars):**
```
Sovereign, air-gapped AI that turns terabytes of seized evidence into court-ready
leads for child-protection cases — 9 months of triage in 47 minutes, with proof
for every claim.
```

**Inspiration (short):**
```
Child-protection investigators drown in seized digital evidence — terabytes per
case, 6–18 month backlogs — while a child waits, and reviewing the material
inflicts real trauma. We asked: what if AI did the triage, and handed a court
proof it could actually trust?
```

**What it does:**
```
AEGIS-X ingests seized devices, seals every file with a cryptographic +
post-quantum fingerprint, and runs four AI lanes in parallel to triage 480,000
files in minutes. Investigators ask the evidence in plain English and get cited
answers; a graph neural network surfaces hidden suspects; a deepfake shield
flags AI-generated media; and one click compiles a court-ready, §63-certified
report where every AI claim is hallucination-scored and traceable to sealed
evidence. It's sovereign and air-gapped — data never leaves the building.
```

**How we built it:**
```
React 19 + Vite front end, TailwindCSS design system, and a Three.js
(react-three-fiber) 3D "Fusion Vault" that renders each case as a stack of
evidence strata. A guided Demo Mode drives the full journey. The architecture
maps to real techniques — hybrid vision transformers, GraphRAG on a local LLM,
GNN link prediction, 3-stream synthetic detection, and BSA-2023 §63 reporting.
```

**Challenges / What we learned / What's next:**
```
Challenges: making a Wizard-of-Oz AI feel genuinely production-grade, and
keeping a 3D world performant and demo-continuous. Learned: trust is a UX
problem — citations, confidence scores, and provenance sell an AI to a courtroom
more than accuracy alone. Next: real model integration inside confidential-GPU
enclaves, cross-agency federated learning, and a field-edge intake kit.
```

**Built with:** `react` `vite` `three.js` `react-three-fiber` `tailwindcss` `graphrag` `gnn` `rag`
